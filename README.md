# 🐕 콩고물 톡

**콩고물 톡은 현재 대화 중인 캐릭터와 RP 밖에서 채팅할 수 있는 SillyTavern 어시스턴트 확장입니다.**

## ⚙️ 확장 설정

**Extensions → 콩고물 톡**에서 연결 API와 디버그 로그를 관리합니다.

| 설정 | 설명 |
|---|---|
| **콩고물 톡 전용 API** | Connection Manager에 저장된 연결 프로필을 선택합니다. 비워두면 현재 메인 API를 사용합니다. |
| **프로필 목록 새로고침 `↻`** | 저장된 연결 프로필 목록을 다시 불러옵니다. |
| **🐞 디버그 로그** | 최근 요청과 오류 정보를 확인하고, 로그를 복사하거나 비웁니다. |

선택한 전용 API 프로필을 찾을 수 없으면 메인 API로 자동 변경하지 않고 오류를 표시합니다.

## 💬 콩고물 톡 패널

확장 메뉴의 **콩고물 톡**을 누르면 패널이 열립니다.

**패널 상단**

| 버튼 | 기능 |
|---|---|
| `—` | 패널을 작은 버튼으로 접습니다. |
| `⚙` | 패널 설정을 엽니다. |
| `×` | 패널을 닫습니다. |
| 접힌 테마 아이콘 | 접어둔 패널을 다시 펼칩니다. |
| 오른쪽 아래 모서리 | 패널 크기를 조절합니다. |

**대화방 관리**

| 버튼 | 기능 |
|---|---|
| **현재 방 이름** | 현재 캐릭터의 대화방 목록을 엽니다. |
| `⋯` | 새 방, 고정, 삭제 버튼을 펼치거나 접습니다. |
| `＋` | 모드를 선택해 새 대화방을 만듭니다. |
| `📌` | 현재 방을 목록 상단에 고정하거나 해제합니다. |
| `🗑️` | 현재 방을 삭제합니다. |

대화방은 캐릭터별로 따로 저장되며, 한 캐릭터에게 여러 모드의 방을 만들어 사용할 수 있습니다.

- 방 목록에서 이전 대화방으로 이동할 수 있습니다.
- 방 이름을 우클릭하거나 길게 누르면 이름을 바꿀 수 있습니다.
- 각 방의 모드, 메시지와 고정 상태가 그대로 유지됩니다.
- 모든 방을 삭제하면 기본 콩톡 방이 다시 만들어집니다.

## 🗂️ 대화 모드

| 모드 | 설명 |
|---|---|
| **콩톡** | RP 밖에서 현재 캐릭터와 자유롭게 대화하는 기본 모드입니다. |
| **집사 모드** | 캐릭터에게 집사 역할을 부여해 일정 정리, 질문, 선택지 비교, 생각 정리와 각종 어시스트 요청에 실용적인 답변을 받습니다. |
| **펫 모드** | 캐릭터에게 사용자의 펫 역할을 부여합니다. 캐릭터의 원래 성격과 말투가 펫 역할에 그대로 반영되어, 순순히 따르거나 투덜대거나 제멋대로 구는 방식도 캐릭터마다 달라집니다. |
| **직장 동료 모드** | 캐릭터에게 같은 직장 동료 역할을 부여해 실제 업무 문구, 판단, 계획, 초안과 다음 할 일을 함께 처리합니다. |
| **RP 어시 모드** | 현재 캐릭터와 RP 밖에서 장면 상황, 감정선, 다음 전개와 답변 방향을 상의합니다. OOC는 직접 요청했을 때 작성합니다. |
| **평행우주AU(Watching RP)** | 원래 캐릭터와 같은 이름, 얼굴, 나이, 성격과 말투를 가진 현대 AU 인물과 메인 RP를 작품처럼 함께 보며 대화합니다. |
| **평행우주AU(Watching RP)🙃** | 원래 캐릭터와 같은 이름, 얼굴과 나이를 가졌지만 성격과 말투는 반대인 현대 AU 인물과 메인 RP를 함께 봅니다. |

새 대화방을 만들면 선택한 모드에 맞는 시작 메시지가 자동으로 준비됩니다.

- **콩톡**은 현재 테마의 시작 아이콘으로 열립니다.
- **집사 · 펫 · RP 어시 모드**는 캐릭터가 부여된 역할에 자기 방식으로 반응하며 시작합니다.
- **직장 동료 모드**는 자연스러운 첫 대화로 시작합니다.
- **평행우주AU**는 새로 정해진 AU 설정과 현재 RP 장면에 대한 반응으로 시작합니다.

## ✉️ 대화하기

| 조작 | 기능 |
|---|---|
| **메시지 입력 + 전송 버튼 클릭** | 현재 방에서 캐릭터의 답변을 생성합니다. |
| **빈 입력창 + 전송 버튼 클릭** | 직전 대화를 이어서 답변을 생성합니다. |
| **AI 답변 생성 중 전송 버튼 클릭** | 생성 중인 답변을 중지합니다. |

**메시지 메뉴**

메시지를 우클릭하거나 길게 누르면 메뉴가 열립니다.

| 메뉴 | 기능 |
|---|---|
| **복사** | 메시지 내용을 복사합니다. |
| **RP 반영** | 최근 콩고물 톡 대화를 메인 RP에 전달할 OOC로 정리합니다. |
| **삭제** | 선택한 메시지를 삭제합니다. |

**RP 반영**

콩고물 톡의 대화는 메인 RP와 따로 저장되며 자동으로 섞이지 않습니다. **RP 반영**을 누르면 현재 방의 최근 메시지 10개를 영어 OOC로 정리해 메인 채팅 입력창에 넣습니다. 내용은 자동 전송되지 않으므로 확인하거나 고친 뒤 직접 보내면 됩니다.

## 🎛️ 패널 설정

패널 상단의 `⚙`에서 현재 캐릭터와 콩고물 톡 화면에 적용할 설정을 조절합니다.

**화면 설정**

| 설정 | 설명 |
|---|---|
| **테마** | 콩고물, 초코딸기, 메론소다, 심플 맥북 중에서 선택합니다. |
| **스크롤 바 고정** | 테마에 맞춘 고정 스크롤바와 브라우저 기본 스크롤바를 전환합니다. |
| **유저 콩톡 닉네임** | 유저 메시지 옆에 표시할 이름이나 이모지를 정합니다. 화면에는 최대 4글자까지 표시됩니다. |
| **채팅창 폰트 크기** | 콩고물 톡 메시지의 글씨 크기를 조절합니다. |

**캐릭터와 업무 설정**

| 설정 | 설명 |
|---|---|
| **캐릭터 말투 고정 메모** | 모든 모드에서 지켜야 할 말투, 금지 표현과 답변 성향을 적습니다. |
| **직장 동료 업무 메모** | 직장 동료 모드가 참고할 실제 직업, 업무와 필요한 도움을 적습니다. |

**반영 범위 설정**

캐릭터 카드, 유저·페르소나 정보, 현재 콩고물 톡 방의 이전 대화와 말투 고정 메모는 기본으로 반영됩니다. 아래 항목도 기본값은 모두 반영하도록 설정되어 있으며, 필요에 따라 포함 여부와 범위를 조절할 수 있습니다.

| 설정 | 조절하는 내용 |
|---|---|
| **프리셋 포함** | 현재 프리셋을 답변에 반영할지 정합니다. |
| **로어북 포함** | 활성 로어북과 월드 인포를 반영할지 정합니다. |
| **확장 메모리 포함** | SillyTavern이나 다른 확장이 제공하는 메모리·요약 정보를 반영할지 정합니다. |
| **최근 RP 읽을 메시지 수** | 캐릭터의 말투, 관계성과 현재 상황을 참고할 메인 RP 범위를 정합니다. |
| **콩톡 기억 범위** | 답변할 때 참고할 현재 대화방의 이전 메시지 수를 정합니다. |
| **답변 목표 토큰 수** | 콩고물 톡 답변의 목표 길이를 조절합니다. |

## 🎨 테마

| 테마 | 특징 |
|---|---|
| **🐕 콩고물** | 인절미·콩고물·밀크티 색감을 담은 포근한 기본 메신저 테마입니다. |
| **🍓 초코딸기** | 딸기 초콜릿 색감을 담은 달콤하고 통통 튀는 테마입니다. |
| **🥤 메론소다** | 메론소다의 청량하고 귀여운 분위기를 담은 테마입니다. |
| **⌘ 심플 맥북** | 맥북 감성을 담은 기호 중심의 깔끔한 흑백 테마입니다. |

테마를 바꾸면 패널과 전송 버튼, 접힌 버튼과 시작 아이콘도 함께 바뀝니다.

## 💾 백업과 저장

| 기능 | 설명 |
|---|---|
| **이 캐릭터 대화 내보내기** | 현재 캐릭터의 모든 대화방과 메시지를 JSON 파일로 저장합니다. |
| **이 캐릭터 대화 가져오기** | 백업 파일을 불러와 현재 캐릭터의 콩고물 톡 데이터를 교체합니다. |
| **모든 대화 내역 초기화** | 현재 캐릭터의 모든 콩고물 톡 대화방과 메시지를 초기화합니다. |

대화방과 메시지는 SillyTavern 서버의 확장 설정에 캐릭터별로 저장됩니다. 같은 서버와 같은 사용자로 접속하면 폰과 컴에서 같은 내역을 사용하며, 메인 채팅방을 옮겨도 같은 캐릭터라면 그대로 유지됩니다. 이전 버전의 브라우저 로컬 사본은 더 이상 읽거나 병합하지 않습니다.

패널 위치와 크기, 테마, 말투 메모, 업무 메모와 반영 범위 설정도 유지됩니다.
